# 使い方
1. [ここ](https://dash.nocaptchaai.com/home)からAPIキーを取得（最初の100~200回は無料）
2. `pip install requests pyppeteer`
3. main.py, solve.pyを編集（email, APIKEYなど）
4. `py main.py`
5. Let's enjoy!!
